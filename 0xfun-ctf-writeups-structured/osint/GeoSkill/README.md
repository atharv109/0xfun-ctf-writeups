# GeoSkill

**Category:** OSINT  
**Source:** acn1.xyz  
**Author:** acni

## Challenge Summary

This was a seven-location GeoGuessr-style challenge. The solve relied on geographic clues, reverse image search, Google Street View meta clues, and general GeoGuessr pattern recognition.

## Location 1

The image showed Christ the Redeemer in Rio de Janeiro, Brazil. This was solvable with reverse image search or by recognizing the statue directly.

## Location 2

A bank visible in the image pointed to Lagos, Nigeria. Searching the bank name led to the city.

## Location 3

Reverse image search gave Christmas Island almost immediately.

## Location 4

The intended region was Ehime Prefecture, Japan. The strongest clue was the style of blue cycling lane markings, which are especially associated with Ehime in GeoGuessr reference material.

## Location 5

This was the hardest location. The solve relied on identifying the Google Street View car model and matching it using a GeoGuessr vehicle reference list.

**Answer:** `Kongebrovej`

## Location 6

The clue referenced the Grand Canyon. Following the river in Street View led to the final location.

**Location:** `///sailor.cascading.mower`

## Location 7

The final location was Bermuda. Clues included left-side driving, English-language environment, and a visible copyright date that narrowed the result dramatically.
