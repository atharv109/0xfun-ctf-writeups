# Insanity Revenge

**Category:** OSINT  
**Source:** krauq  
**Flag:** `0xfun{0m_built_a5_a_G1F}`

## Solve Flow

1. Use the same Discord server context as Insanity 1.
2. Inspect the custom animated emoji.
3. Extract its frames.
4. Notice that one nearly invisible frame contains the flag.

The challenge relied on players missing the second frame because the visible logo frame dominated the animation.
